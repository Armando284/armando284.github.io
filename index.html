About COLOR GENERATOR -v1.0.1

During my last project I found myself working with a lot of charts that used a dinamic amount of datasets. I needed to color each dataset dinamicaly with vibrant colors wich wouldn't look bad when combined with the others, and since I wouldn't know how many datasets there would be I couldn't set those colors from a palette before hand. At the time I couldn't find a framework or module for that, so I decided to code it myself. I researched about color theory, web color palette designs and the way to randomize colors html codes. After some work I decided to create couples of random complementary colors since they look good on charts. The first version I made was writen on php. Then I thought it may be better to write it on javascript and publish it in case someone has the same need as I did, as an extra I decided to add some functions to get other standar color palettes and download those as css. This is the result. I hope you find it usefull and I'll keep updating it with more colors and functions on my free time.

The code is divided in modules. A color module, owns the logic for the diferent levels of saturation of each color and how to randomize it. For example the method to randomize levels of color: 

              
                  /*
                  *	 Recives a minimun and maximun value.
                  *  Returns a random int between those values.
                  */
                  randomInt(min, max) {
                    return Math.floor(Math.random() * (max - min) + min);
                  }

Then each color class uses it to make random colors inside of its palette range: 

              
                  /*
                  *  Returns an object with a random saturated red color
                  *  and a label for understanding.
                  */
                  sat() {
                    this.color = "Saturated red";
                    //rojo (Rgb)
                    this.rgb[0] = this.randomInt(196, 232);
                    //verde (rGb)
                    this.rgb[1] = this.randomInt(0, 1);
                    //azul (rgB)
                    this.rgb[2] = this.randomInt(40, 48);

                    return {
                      color: this.color,
                      rgb: this.rgb
                    };
                  }

Then we use this method to convert decimal rgb colors into hexadecimal for html: 

              
                  /*
                  *  Recives an array of 3 positions, each for a level of an rgb color.
                  *  Returns an string with the color hexadecimal code.
                  */
                  RGBtoHTML(rgb) {
                    let html = "#";
                    for (let i = 0; i < 3; i++) {
                      if (rgb[i] < 16) {
                        html = `${html}0${rgb[i].toString(16)}`;
                      } else {
                        html = `${html}${rgb[i].toString(16)}`;
                      }
                    }
                    return html;
                  }

And with this method we create a functional palette in the base of 12 colors, the bigger the base the more colors we will have: 

              
                  /*
                  *  Recives an index of an array, so the color will mix well with others araound it.
                  *  Returns an object with an string with the color hexadecimal code and a counter.
                  */
                  complementary(index) {
                    let color = "";
                    let orden = 0;
                    if (index == 0 || index % 12 == 0) {
                      //Rojo saturado
                      color = this.RGBtoHTML(this.RED.sat().rgb);
                      orden = 1;
                    } else if (index % 12 == 1) {
                      //Verde no saturado
                      color = this.RGBtoHTML(this.GREEN.dark().rgb);
                      orden = 2;
                    } else if (index % 12 == 2) {
                      //Azul saturado
                      color = this.RGBtoHTML(this.BLUE.sat().rgb);
                      orden = 3;
                    } else if (index % 12 == 3) {
                      //Naranja no saturado
                      color = this.RGBtoHTML(this.ORANGE.dark().rgb);
                      orden = 4;
                    } else if (index % 12 == 4) {
                      //Amarillo saturado
                      color = this.RGBtoHTML(this.YELLOW.sat().rgb);
                      orden = 5;
                    } else if (index % 12 == 5) {
                      //Violeta no saturado
                      color = this.RGBtoHTML(this.VIOLET.dark().rgb);
                      orden = 6;
                    } else if (index % 12 == 6) {
                      //Verde-lima saturado
                      color = this.RGBtoHTML(this.LIME.sat().rgb);
                      orden = 7;
                    } else if (index % 12 == 7) {
                      //Morado no saturado
                      color = this.RGBtoHTML(this.PINK.dark().rgb);
                      orden = 8;
                    } else if (index % 12 == 8) {
                      //Ocre saturado
                      color = this.RGBtoHTML(this.OCHRE.sat().rgb);
                      orden = 9;
                    } else if (index % 12 == 9) {
                      //Prusia no saturado
                      color = this.RGBtoHTML(this.PRUSSIAN.dark().rgb);
                      orden = 10;
                    } else if (index % 12 == 10) {
                      //Salmon saturado
                      color = this.RGBtoHTML(this.SALMON.sat().rgb);
                      orden = 11;
                    } else if (index % 12 == 11) {
                      //Azul-cielo no saturado
                      color = this.RGBtoHTML(this.SKY.dark().rgb);
                      orden = 12;
                    }
                    return {
                      color: color,
                      order: orden,
                    };
                  }
